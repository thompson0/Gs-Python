#Jogo feito com pygame 

1. **Criar um ambiente virtual**:
   `python -m venv venv`

2. **Ativar o ambiente virtual**:
   - No Windows:
     `venv\Scripts\activate`
   - No Linux/MacOS:
     `source venv/bin/activate`

3. **Instalar o pygame**:
   `pip install pygame`
4. **Iniciar o jogo**:
 `  python jogo.py`
